# Piton IDE 🐍
Bahasa Pemrograman Indonesia berbasis Python, berjalan di browser!

## ✨ Cara Pakai
1. Buka link GitHub Pages: `https://[username].github.io/piton-ide`
2. Ketik kode Piton di editor.
3. Klik tombol **Jalankan** untuk melihat hasilnya.

## 📖 Kamus Piton
| Kategori | Piton | Python |
|----------|-------|--------|
| I/O | `ketik()` | `print()` |
| I/O | `terima()` | `input()` |
| Fungsi | `buat` | `def` |
| Logika | `jika` | `if` |
| ... dan lainnya ada di tabel dalam aplikasi |

## 🛠️ Cara Edit / Clone
1. Fork repo ini.
2. Clone ke komputer/HP kamu.
3. Edit file `index.html`.
4. Commit dan push perubahan.
